# 🎓 Student Admission Form GUI (Python Project)

## 📌 Project Overview

This project is a **Graphical User Interface (GUI) based Student Admission Form** developed using **Python (Tkinter)**. It allows users to enter student details in an organized format and stores the data automatically in an Excel file.

The aim of this project is to simplify the **student admission process** by replacing manual paperwork with a digital solution.

---

## 🚀 Features

* 🧑‍🎓 Collects complete student details:

  * Personal Information (Name, DOB, Gender)
  * Contact Details (Address, Email, Phone)
  * Parent/Guardian Information
  * Academic Details (Class 12 Marks)
* 📊 Automatically calculates **PCM aggregate percentage**
* 📁 Stores data in an **Excel file (.xlsx)**
* 🖥️ User-friendly GUI using Tkinter
* 🔄 Clear and Reset form functionality
* 📜 Scrollable interface (handles large forms)

---
